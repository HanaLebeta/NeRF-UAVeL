NeRF-UAVeL: Unified Attention-driven Volumetric Learning for Enhanced NeRF-based 3D Object Detection

Author 11, Author 21, Author 32
1Affiliation 1 2Affiliation 2
Paper Code Data arXiv

Abstract

This paper presents NeRF-UAVeL, a unified attention-driven volumetric learning framework for 3D object detection from Neural Radiance Fields. The framework integrates four complementary modules into a volumetric backbone to enhance multi-scale feature representation and spatial context modeling:

(1) Multi-dimensional Volumetric Attention Pooling (MVAP), which decomposes volumetric attention into axis-specific components to capture directional feature asymmetry; (2) Tri-Scale Asymmetric Convolutional Aggregation (TACA), which employs spatial-first, depth-second factorized convolutions for multi-scale feature extraction; (3) Dual-Domain Attention Fusion (DDAF), which jointly recalibrates channel and spatial features through ECA-based attention and multi-rate dilated convolutions; and (4) Volumetric Cross-Window Attention Fusion (V-CWAF), which applies localized cross-window multi-head attention for global context modeling.

Extensive experiments on the 3D-FRONT and ScanNet datasets demonstrate that NeRF-UAVeL achieves state-of-the-art performance, with improvements of +6.7% AP50 on 3D-FRONT and +6.9% AP50 on ScanNet over the NeRF-RPN baseline.

Proposed Modules

Four complementary attention-driven modules designed to address different aspects of volumetric feature learning.

MVAP

Multi-dimensional Volumetric Attention Pooling

Decomposes volumetric attention into axis-specific components along height, width, and depth dimensions. This captures directional feature asymmetry inherent in 3D scenes, enabling the network to learn dimension-aware feature representations that reflect the anisotropic nature of indoor environments.

MVAP Diagram

TACA

Tri-Scale Asymmetric Convolutional Aggregation

Employs spatial-first, depth-second factorized convolutions at three scales to extract multi-scale features efficiently. The asymmetric decomposition reduces computational cost while preserving the ability to capture both fine-grained local details and broader contextual patterns across volumetric space.

TACA Diagram

DDAF

Dual-Domain Attention Fusion

Jointly recalibrates features in both channel and spatial domains through ECA-based channel attention and multi-rate dilated convolutions. This dual-domain approach ensures that the network can simultaneously emphasize informative feature channels and spatially relevant regions for accurate detection.

DDAF Diagram

V-CWAF

Volumetric Cross-Window Attention Fusion

Applies localized cross-window multi-head attention to model long-range dependencies across the volumetric feature space. By partitioning volumes into windows and attending across their boundaries, V-CWAF achieves global context modeling with linear complexity relative to volume size.
